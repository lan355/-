Программа осуществляет запись и хранение информации о государствах в базе данных.

Структура меню программы:

Добавить запись в базу
Просмотр Базы Данных
Сортировать
-> Сортировать по названию
-> Сортировать по населению
-> Сортировать по площади
-> Назад
Изменить
-> Изменить название
-> Изменить столицу
-> Изменить континент
-> Изменить численность
-> Изменить площадь
-> Назад
Удалить
Дополнительные функции
-> Удалить, если численность населения меньше заданого
-> Поиск по названию столицы
-> Поиск занимающей площади
-> Назад
Изменить/Создать пароль
Справка
Выход


Добавить запись в базу - Добавляет новую запись в базу данных
Просмотр Базы Данных - Позволяет просмотреть записи в базе данных
Сортировать - Открывает меню сортировки по различным данным
Сортировать по названию - Осуществляет сортировку по названию страны
Сортировать по населению - Осуществляет сортировку по населению страны
Сортировать по площади - Осуществляет сортировку по площади страны
Назад - Возвращает главное меню
Изменить - Изменяет запись в базе данных
Удалить - Удаляет запись в базе данных
Дополнительные функции - Открывает меню дополнительных функций
Удалить, если численность населения меньше заданого - Удаляет все записи в базе данных 
	численность населения в которых меньше заданной
Поиск по названию столицы - Оуществляет поиск по названию столицы
Поиск занимающей площади - Осуществляет поиск по занимаемой площади
Изменить/Создать пароль - Создает начальный или заменяет существующий пароль
Выход - Завершает работу программы
